NMS環境

1. 安裝 node.js 6.6.0
2. 安裝 npm install node-gyp -g
3. 安裝 npm install node-pre-gyp -g
4. 安裝 npm install nw-gyp -g
5. 安裝 npm install enclose -g
6. 安裝 python 2.7.x, 不支援3.x, 要設定環境變數
7. 安裝 windows SDK 7.1 => 需要先移除 visual studio C++ 2010 redistributable
8. 安裝 visual sutio express 2013
9. 安裝 windows SDK 64bit
10.安裝 7zip

編譯sqlite3 for node-webkit
#node-pre-gyp rebuild --runtime=node-webkit --target=0.17.4 --msvs_version=2013 --target_arch=x64
// 32 bit => ia32 --target 為nw.js版本, 目前為0.17.4

編譯device package
#code_path\pack\build_device_package.bat
pack device package
#code_path\pack\zip_device_package_win_x64.bat
檔案產生在code_path\dni_smartview\pack\GenerateDevicePackage\build\

pack NMS
#code_path\buils_js.bat
